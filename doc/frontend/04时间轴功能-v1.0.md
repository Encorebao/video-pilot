# 时间轴功能 v1.0

## 概述

时间轴是编辑器的核心操作区。v1.0 实现了可用于实际剪辑的基础交互集合，所有操作基于前端状态（Zustand），通过 Remotion Player 实时反映到预览区。

---

## 功能清单

### 1. 轨道与片段显示

- 支持多条**视频轨**与多条**音频轨**并排展示
- 每条轨道左侧显示轨道名称标签
- 片段按 `sourceType` 区分颜色：
  - `imported-video`：蓝色
  - `extracted-audio`：灰白
  - `tts`：紫色
  - `recording`：浅灰
  - `music`：绿色
- 片段内显示标题文字，超出宽度时裁剪
- 音频类片段内显示 **SVG 波形**，基于 Web Audio API 解码音频文件，600 采样点绘制包络路径 + 高亮边缘线

### 2. 标尺与时间码

- 顶部标尺按当前缩放级别自动计算刻度间隔，显示 `MM:SS` 格式时间标注
- 播放头以红色竖线 + 顶部三角形显示，覆盖标尺与轨道区域

### 3. 播放头

- 点击标尺任意位置：跳转到对应帧
- 点击轨道空白区域：跳转到对应帧
- 播放头位置与 `useTimelineStore.currentFrame` 双向同步，联动 Remotion Player

### 4. 缩放

- 工具栏右侧缩放滑块：拖动调节缩放倍率（0.25× ～ 10×）
- `Ctrl/Cmd + 鼠标滚轮`：在时间轴区域内缩放，以鼠标位置为中心（正在实现中）
- `+` / `-` 图标按钮：步进调整缩放

### 5. 片段选中

- 点击片段：选中（高亮边框 + 发光效果），同时更新 `selectedClipId` / `selectedTrackId`
- 点击空白区域：取消选中
- 未选中状态：片段边框使用对应颜色的低不透明度样式
- 已选中状态：片段边框高亮，四角显示操作把手

### 6. 片段移动（拖拽）

- 拖拽片段主体：横向移动片段在时间轴上的位置
- **吸附（Snap）**：拖拽时自动吸附到其他片段的起始帧或结束帧（8px 容差）
- 移动结果写回 `project-store.moveClip`

### 7. 修剪（Trim）

- 片段两侧显示修剪把手（选中时白色发光，未选中时灰色低调）
- **左修剪**（拖动左把手）：
  - 更新片段的 `startFrame` 和 `durationInFrames`
  - 同步更新 `sourceIn`（源文件入点），确保播放内容从正确帧开始
  - 写回 `project-store.trimClipStart`
- **右修剪**（拖动右把手）：
  - 只更新 `durationInFrames`
  - 写回 `project-store.trimClipEnd`
- 波形 SVG 随修剪实时裁剪显示范围（viewBox 动态计算），不拉伸

### 8. 分割

- **工具栏"分割"按钮**（需先选中片段）：
  - 在当前播放头位置（`currentFrame`）将选中片段切为两段
  - 左半：保留原 `startFrame` 和 `sourceIn`
  - 右半：`startFrame = currentFrame`，`sourceIn` 相应推进
  - 播放头不在片段范围内时操作无效
  - 操作后自动取消选中状态
- 写回 `project-store.splitClip`

### 9. 删除

- **工具栏"删除"按钮**（需先选中片段）：
  - 从对应轨道中移除选中片段
  - 操作后自动取消选中状态
- 写回 `project-store.deleteClip`

### 10. 工具栏布局

工具栏分为三个区域：

| 区域 | 内容 |
|------|------|
| 左 | 删除、分割（未选中片段时置灰禁用） |
| 中 | 跳转到起点、播放/暂停、跳转到终点、时间码显示 |
| 右 | 缩小（−）、缩放滑块、放大（+） |

### 11. 与 Remotion Player 联动

- `currentFrame` 变化时同步更新 Player 的 `seekTo`
- 播放中 Player 的 `onTimeUpdate` 回调反向更新 `currentFrame`，带防抖保护避免循环
- 片段使用 `<Sequence from={startFrame} durationInFrames={...}>` 包裹
- 视频片段使用 `<Video startFrom={sourceIn}>` 保证修剪后从正确帧播放
- 音频片段使用 `<Audio startFrom={sourceIn}>` 同理

## 数据模型

```ts
interface TimelineClip {
  id: string;
  mediaId: string;        // 关联 MediaItem
  title: string;
  startFrame: number;     // 在时间轴上的起始帧
  durationInFrames: number;
  sourceIn: number;       // 源文件入点（左修剪后更新）
  color: string;
  sourceType: "imported-video" | "extracted-audio" | "tts" | "recording" | "music";
}
```

## 已知限制（v1.0 范围内暂不处理）

- 片段不支持纵向跨轨拖拽
- 不支持多选
- 吸附仅处理片段边缘，不处理播放头吸附
- 分割后两段 id 以 `-L` / `-R` 后缀区分，不做持久化 id 管理
- 移动时不检测片段重叠
