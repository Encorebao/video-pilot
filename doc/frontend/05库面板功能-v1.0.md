# 库面板功能文档 v1.0

> 对应左侧侧边栏 **库** Tab（原"素材"）

---

## 一、面板概览

库面板是项目素材的统一管理入口，用户可以在此导入、浏览、筛选、排序素材，并直接触发单项或全量 AI 分析。面板宽度默认 384px，可通过拖拽调整（160–640px）。

---

## 二、功能清单

### 2.1 素材导入

| 操作 | 描述 |
|---|---|
| 点击"导入"按钮 | 弹出导入方式选择菜单 |
| 导入到项目 | 将文件复制到项目目录统一管理 |
| 保持原位引用 | 文件保留在原路径，项目只记录引用关系 |
| 确认方式后 | 弹出系统文件选择器，支持多选，接受 `video/*` 和 `audio/*` |

导入后自动记录：
- `createdAt`：导入时间（ISO 8601）
- `updatedAt`：文件最后修改时间（来自 `file.lastModified`）
- `fileSize`：文件大小（字节）

### 2.2 素材筛选

工具栏提供三个筛选 Tab，互斥选择：

| Tab | 显示内容 |
|---|---|
| 全部 | 所有素材 |
| 视频 | `type === "video"` |
| 音频 | `type === "audio"` 或 `type === "generated-audio"` |

### 2.3 排序

点击 **ArrowUpDown 排序按钮**弹出下拉菜单，支持以下排序键：

| 排序键 | 说明 |
|---|---|
| 名称 | 按文件名中文本地化排序 |
| 文件类型 | 视频 → 音频 → 生成音频 → 字幕 |
| 创建时间 | 按 `createdAt` ISO 字符串比较 |
| 修改时间 | 按 `updatedAt`（缺省回退到 `createdAt`） |
| 文件大小 | 按 `fileSize` 数字升序/降序 |

- 默认排序键：创建时间，升序
- 再次点击同一排序键可切换升序/降序，当前方向以 `↑` / `↓` 标示

### 2.4 素材卡片

每条素材以卡片形式展示：

| 元素 | 内容 |
|---|---|
| 缩略图 | 带类型色彩的图标（视频蓝、音频绿） |
| 文件名 | 截断展示，hover 加亮 |
| 副文字 | 类型标签 · 时长（如 `8.9s`，超过 60s 则 `m:ss`） |
| 激活态 | 选中时显示环形高亮 + "预览中" 标签 |

**点击卡片**：切换进入/退出 **分屏预览模式**（左侧显示该素材播放器，右侧为项目合成播放器）。

### 2.5 AI 分析

#### 全库分析
- 位置：面板标题栏"库"右侧
- 图标：`Sparkles`（分析中变 `Loader2` 旋转 + "分析中"文字）
- 行为：对所有当前项目素材逐一触发分析任务

#### 单项分析
- 位置：素材卡片右侧，**hover 时显示**
- 图标：`Sparkles`（分析中变 `Loader2`）
- 分析中：缩略图叠加半透明遮罩 + 紫色旋转 spinner
- 分析完成：
  - 缩略图右上角出现绿点（`bg-emerald-400/80`）
  - 副文字末尾追加"已分析"绿色标签
  - Sparkles 图标变绿，可再次触发重新分析

**当前实现为 Mock**：setTimeout 模拟 1.4–2.2s 延迟，后续对接后端时替换 `useUIStore.analyzeItem()` 内部实现即可。

## 三、数据模型

```ts
interface MediaItem {
  id: string;
  name: string;
  type: "video" | "audio" | "generated-audio" | "caption";
  importMode: "copied" | "referenced";
  originalPath: string;
  projectPath?: string;       // 仅 copied 模式有值
  durationInFrames: number;
  sourceLabel: string;
  createdAt?: string;         // ISO 8601
  updatedAt?: string;         // ISO 8601，文件修改时间
  fileSize?: number;          // 字节
}
```
