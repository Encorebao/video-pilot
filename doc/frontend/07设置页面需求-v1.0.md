# 设置页面需求 v1.0

路由：`/settings`  
状态：UI 已实现，交互全部为前端本地状态，暂无持久化。

---

## 页面结构

```
┌─────────────────────────────────────────────────────┐
│  ← 返回编辑器  ·  设置                               │  ← 顶栏 (h-11)
├──────────────┬──────────────────────────────────────┤
│              │                                      │
│  左侧导航    │  内容区域                             │
│  (w-52)      │  (max-w-xl, 居中)                    │
│              │                                      │
└──────────────┴──────────────────────────────────────┘
```

左侧导航分为两个分组，点击导航项切换右侧内容（SPA 无路由跳转）。

## 系统设置

### 1. 外观与语言

| 设置项 | 控件类型 | 默认值 | 说明 |
|--------|----------|--------|------|
| 界面语言 | Select | 简体中文 | 重启后生效；选项：简体中文 / English / 日本語 |
| 主题 | Select | 深色 | 深色 / 浅色 / 跟随系统 |

## AI 模型

> 第三个导航分组，数据来自 `useSettingsStore`（`modelConfigs` + `dependencies`）。

### 9. 模型接入

展示所有 AI 模型卡片，按类别顺序排列：VL → LLM → Audio → TTS。

每张模型卡包含：

| 元素 | 说明 |
|------|------|
| 模型名称 + 供应商 | 标题行 |
| 类别标签 | 视觉理解 / 语言模型 / 语音识别 / 语音合成 |
| 状态徽章 | 已检查（emerald）/ 已配置（blue）/ 未配置（dim） |
| 描述文本 | 模型用途简介 |
| API 地址输入框 | 全宽文本输入 |
| API Key 输入框 | 密码类型 + 「显示/隐藏」切换按钮 |
| 保存配置按钮 | 仅在内容有变更时可用（`isDirty`） |
| 连通性检查按钮 | 调用 `runModelCheck`，更新 `lastCheckedAt` |
| 帮助文档链接 | 外链，打开新标签页 |
| 最近检查时间 | 仅在 `lastCheckedAt` 存在时显示 |

### 10. 依赖检查

展示系统依赖项列表（ffmpeg、ffprobe、Python Runtime、funasr、Remotion）。

每张依赖卡包含：

| 元素 | 说明 |
|------|------|
| 依赖名称 | 标题 |
| 类型标签 | 系统工具 / Python 运行时 / Python 包 |
| 状态徽章 | 已安装（emerald）/ 缺失（red）/ 需确认（amber） |
| 描述 | 用途说明 |
| 安装命令 | 代码块样式，含当前版本 |
| 操作按钮 | 已安装：「刷新状态」；缺失/需确认：「标记为已安装」 |
